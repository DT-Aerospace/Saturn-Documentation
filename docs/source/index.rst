Welcome to the Saturn Documentation!
===================================

**Saturn** is a Avionics system used by DT Aerospace onboard our model rockets
it is feature rich and provides all the functionaility and tracking necessery for both safe and informative flights.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development and the documentation will be updated regurlarly.
